# Hunyuan3D-2.1 runtime runner

This runner executes the official image-to-shape pipeline followed by the official PBR paint pipeline.

Hunyuan3D-2.1 outputs are restricted by the Tencent Hunyuan Community License and must not be served to EU27, GB, or KR regions. Batch C cloud execution is expected to use a non-EU RunPod data center.
